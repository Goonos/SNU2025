3장

- 이벤트란
  웹 브라우저 내에서 사용자가 실행하는 동작
  이벤트 발생 시 처리할 동작이 있어야 하며, 이를 위해 이벤트 핸들러가 필요

- 이벤트 연결 방법
  1. HTML 태그 내 onXXX 속성 사용
    <button onclick="alert('클릭!')">Click</button>
  2. 자바스크립트에서 함수 연결
  3. addEventListener("이벤트명", 함수, 캡처여부)

- 이벤트 버블링과 캡처링
  버블링: 이벤트가 하위 요소에서 상위 요소로 전파
  캡처링: 이벤트가 상위 요소에서 하위 요소로 전파
  addEventListener의 세 번째 인자(true = 캡처, false = 버블)

- 이벤트 객체
  이벤트 발생 시 관련 정보가 담긴 객체 전달
  예) keydown 이벤트에서 e.code, e.key 사용 가능

- 이벤트 예시
  document.body.addEventListener("keydown", (e) => {
    console.log(\`code: \${e.code}, key: \${e.key}\`);
  });

- 기본 동작 취소
  e.preventDefault()로 기본 동작 방지 가능

- 이벤트 전파 중지
  e.stopPropagation()으로 이벤트 전파 중단 가능

7장 : 내용 추가/제거 및 방법

- DOM 요소 추가
  createElement(), appendChild(), insertBefore() 등 사용

- DOM 요소 제거
  removeChild() 사용

- 예시
  const div = document.createElement('div');
  div.textContent = "새로운 내용";
  document.body.appendChild(div);

- 내용 수정
  element.textContent, element.innerHTML 활용

- 스타일 조작
  element.style 속성 사용

- 실습 예시
  버튼 클릭 시 리스트 항목 추가/삭제 구현 가능
