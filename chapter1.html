1주차

- 인라인 스크립트
  HTML 태그 안에 직접 작성하는 자바스크립트
  팝업창을 열고 닫거나, 알림 메시지를 표시할 때처럼 간단한 명령을 처리할 경우 사용

- 내부 스크립트
  <script>/</script> 사이에 실행할 스크립트 소스 작성
  스크립트 태그는 웹문서 모든 곳에 위치할 수 있고, 그 위치에서 바로 실행됨
  한 문서 안에 여러 개의 스크립트 태그 가능
  내부 스크립트는 주로 </body> 앞에 사용

- 인라인, 내부 스크립트의 단점
  모두 내부에서 스크립트를 작성하기 때문에 같은 소스를 여러 웹문서에서 사용할 경우
  같은 소스를 반복해야 하므로 비효율적
  또한 수정하려면 모든 곳을 찾아서 수정해야 함
  따라서 외부 스크립트 파일을 저장하여 링크하는 것이 좋음 (모듈화)

- 외부 스크립트 연결하기
  1. 확장자: .js
  2. HTML 문서에서 스크립트 태그를 사용해서 외부 스크립트 연결
  3. <script src="스크립트 파일경로"></script>
  4. 외부 스크립트 파일에는 스크립트 태그 없이 소스만 작성
  5. 보통 js 폴더를 만들어 저장하는 것이 좋음

- 연결 예시
  a\\b\\c.html 에서 a\\b\\c\\d.js 연결하기
  <body>
    <p>연결하는 방법</p>
    <script src="c/d.js"></script>
  </body>
