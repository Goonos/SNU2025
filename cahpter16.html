웹 API (Web API)

  - 웹 API란?
    웹 브라우저가 제공하는 자바스크립트 API 집합으로, 웹 문서에서 브라우저 기능을 제어하고 활용할 수 있게 한다.
    예) DOM 조작, 위치 정보 조회, 네트워크 통신 등 다양한 기능을 브라우저가 API 형태로 제공.

  - 웹 API 특징
    - 함수와 메서드 형태로 제공되어 웹 페이지에서 호출 가능
    - 비동기 처리, 이벤트 처리 등 다양한 웹 환경 기능을 지원
    - 표준화되어 있어 대부분의 최신 브라우저에서 사용 가능

  - 주요 웹 API 종류
    - DOM API: 문서 구조 조작
    - Geolocation API: 위치 정보 획득
    - Fetch API / XMLHttpRequest: 네트워크 통신
    - Canvas API: 그래픽 처리
    - Web Storage API: 클라이언트 저장소

스토리지 API (Storage API)

  - 스토리지 API란?
    웹 브라우저 내에 데이터를 저장하고 관리할 수 있는 클라이언트 저장 공간 API.
    쿠키보다 용량이 크고 사용이 간편하여 웹 애플리케이션 상태 관리에 많이 사용됨.

  - 주요 스토리지 종류
    1. LocalStorage
      - 영구 저장 (브라우저 종료 후에도 데이터 유지)
      - key-value 쌍 형태 저장
      - 용량은 보통 5MB 정도 제공

    2. SessionStorage
      - 세션 단위 저장 (탭이나 창이 닫히면 데이터 삭제)
      - key-value 쌍 형태 저장

  - 스토리지 사용법 예제

  // LocalStorage에 데이터 저장
  localStorage.setItem('username', 'kim');

  // LocalStorage에서 데이터 읽기
  let user = localStorage.getItem('username');
  console.log(user); // 출력: kim

  // LocalStorage에서 데이터 삭제
  localStorage.removeItem('username');

  // SessionStorage에 데이터 저장
  sessionStorage.setItem('token', 'abc123');

  // SessionStorage에서 데이터 읽기
  let token = sessionStorage.getItem('token');
  console.log(token); // 출력: abc123

  - 스토리지 API 장점
    - 별도의 서버 요청 없이 빠르게 데이터 저장 및 조회 가능
    - 클라이언트 상태 유지에 용이
    - 간단한 문자열 저장 구조로 사용법이 매우 직관적

Geolocation API 예제

  if (navigator.geolocation) {
    navigator.geolocation.getCurrentPosition(
      (position) => {
        console.log(\`위도: \${position.coords.latitude}, 경도: \${position.coords.longitude}\`);
      },
      (error) => {
        console.error('위치 정보 조회 실패:', error);
      }
    );
  } else {
    console.log('Geolocation API를 지원하지 않는 브라우저입니다.');
  }

Fetch API 간단 예제

  fetch('data.json')
    .then(response => response.json())
    .then(data => {
      console.log('데이터:', data);
    })
    .catch(error => {
      console.error('오류 발생:', error);
    });
