9장

- 객체 만들기
  객체는 키:값 쌍 형태로 중괄호{}로 감싼다.

- 생성자 함수
  함수 내 this를 사용해 객체 속성 초기화.

- 객체 키와 값 조회
  for (변수 in 객체)로 키 순회 가능.
  Object.keys(객체) → 키 배열 반환
  Object.values(객체) → 값 배열 반환
  Object.entries(객체) → [키, 값] 배열 반환

- 프로토타입과 상속
  생성자 함수 선언 시 프로토타입 자동 생성.
  공유 속성과 메서드를 프로토타입에 저장.
  프로토타입 상속 통해 자바스크립트 상속 구현.

- 프로토타입 상속 예시
function Textbook(title, price, major) {
  Book.call(this, title, price); // 부모 생성자 호출
  this.major = major;
}
Textbook.prototype.buyTextbook = function() {
  console.log(\`\${this.major} 전공 서적, \${this.title}을 구매했습니다.\`);
};
Object.setPrototypeOf(Textbook.prototype, Book.prototype); // 상속 연결

- 클래스 상속
  class TextbookC extends BookC {
    constructor(title, price, major) {
      super(title, price); // 부모 생성자 호출
      this.major = major;
    }
  }

10장: 문자열 다루기

- 문자열 길이 확인
  let str = "Hello";
  str.length // 5

- 문자 추출
  str.charAt(3) // 3번째 문자
  str[3] // 배열처럼 접근 가능

- 특정 문자 개수 세기
  function counting(str, ch) {
    let count = 0;
    for(let i = 0; i < str.length; i++) {
      if(str[i] === ch) count++;
    }
    return count;
  }

- 문자열 검색
  indexOf("문자열"): 첫 등장 위치, 없으면 -1
  lastIndexOf("문자열"): 마지막 등장 위치
  includes("문자열"): 포함 여부
  startsWith("문자열"), endsWith("문자열"): 시작/끝 검사

- 문자열 자르기
  substring(start, end)
  slice(start, end) — 음수 인덱스 가능
  예) str.substring(5), str.slice(-5)

- 문자열 분할
  str.split("구분자") → 배열 반환
  예) "Hello world".split(" ") → ["Hello", "world"]

- 공백 제거
  str.trim(), trimStart(), trimEnd()

- 대소문자 변환
  str.toUpperCase(), str.toLowerCase()

- 이메일 처리 예시
  • @ 기준으로 분리 후 사용자명과 도메인 추출
  • 사용자명 일부만 보여주고 도메인은 그대로 표시
  • split(), substring() 사용

- 예)
  let email = "username@example.com";
  let parts = email.split("@");
  let user = parts[0].substring(0, 3);
  let domain = parts[1];
  console.log(\`\${user}...@\${domain}\`);
